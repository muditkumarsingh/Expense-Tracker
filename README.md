# 💰 Expense Tracker (MERN)

## 📌 Overview

This is a simple **Expense Tracker** web application built using the
**MERN stack (MongoDB, Express, React, Node.js)**.\
It is one of my first full-stack projects and helped me gain a solid
understanding of how modern web applications work end-to-end.

🔗 **Repository:**
https://github.com/muditkumarsingh/Expense-Tracker.git\
🌐 **Live Demo:** https://expense-tracker-omega-khaki-99.vercel.app/

------------------------------------------------------------------------

## 🚀 Key Learnings

### 🔹 Full Stack Development

-   Understood how **client and server communicate**
-   Learned handling of HTTP methods:
    -   `GET`, `POST`, `PUT`, `PATCH`
-   Gained clarity on **API structure and data flow**

### 🔹 Frontend (React)

-   Worked with **React Router DOM** for navigation
-   Explored **Recharts** for data visualization
-   Learned how to build clean and interactive UI components

### 🔹 Authentication (Most Important 🚨)

-   Implemented **JWT Authentication**
-   Learned:
    -   How **access tokens** and **refresh tokens** are created
    -   Secure handling of tokens
    -   Regenerating access tokens using refresh tokens after expiration

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   **Frontend:** React (Vite), Tailwind CSS, React Router, Recharts\
-   **Backend:** Node.js, Express.js\
-   **Database:** MongoDB\
-   **Authentication:** JWT (Access + Refresh Tokens)

------------------------------------------------------------------------

## ⚙️ Features

-   ➕ Add, update, and delete expenses and income\
-   📊 View all expenses and income clearly **visualized through
    charts**\
-   📁 Download income and expense data as **Excel files**\
-   🔐 Secure authentication system using JWT\
-   🔄 Persistent login sessions using access & refresh tokens

------------------------------------------------------------------------

## ▶️ How to Run the Project

``` bash
# Clone the repository
git clone https://github.com/muditkumarsingh/Expense-Tracker.git

# Setup Backend
cd Expense-Tracker/backend
npm install
npm run dev

# Setup Frontend (in a new terminal)
cd Expense-Tracker/frontend
npm install
npm run dev
```
